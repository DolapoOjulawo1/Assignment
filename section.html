DIFFERENCE BETWEEN PADDINGS, MARGINS AND BORDERS
PADDING
Padding refers to the space between the element content and the element border. It is used to create space around an element's content inside of any defined borders, It is also the inner space between the content and the border of your box. Padding can be set to zero pixels and above,it can change its own element's size and impact how much of a fill color or background image is visible within a box. It controls how much breathing room exists within a box

MARGINS
The margin propery is used to define the space around elements. It clears an area around the element. It is used to create space around elements outside of any defined borders, It is also the outer space (or lack of space) surrounding the box. Margin are essentially invisible barriers around a box. They define how an element interact with adjacent elements or the edge of a screen. Margins can be set to zero and above like padding. But unlike padding, margins can also be set to auto and can even have negative values. It also controls how much breathing room or whitespace exists outside of a box.

BORDERS
Border is used to set the border of an element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of an element's border.It is the perimeter of the box, it can be invisible or they could be a thick coloured line. Border is the strip that goes around your div/element of the content.
